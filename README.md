# Defense Intelligence System

This project analyzes satellite images and sensor data using:

- YOLOv8 (object detection)
- Isolation Forest (anomaly detection)
- XGBoost (threat prediction)
- Gemini AI (report generation)

## Features
- Detects vehicles and objects
- Identifies suspicious patterns
- Predicts high-risk threats
- Generates intelligence reports

## Output
- Dashboard
- CSV analysis
- PDF report
